Lesson 8: Store total attempts with localStorage
Question:
Why does the attempts variable reset after a page reload unless we save it in localStorage?

Student response:
Because regular JavaScript variables only exist in the browser's temporary memory (RAM).
When you reload the page, that memory is completely cleared, resetting the variables. localStorage is needed to save the data to the browser's permanent storage so it survives a reload.
